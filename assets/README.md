# Assets Folder Structure

This folder contains all icons and images used throughout the portfolio website. All icons are in SVG format for scalability and easy customization.

## Folder Structure

```
assets/
├── icons/           # Technology stack icons
├── systems/         # System architecture icons
├── challenges/      # Engineering challenge icons
├── leadership/      # Leadership skill icons
├── ai/              # AI development icons
├── debugging/       # Debugging skill icons
├── customer/        # Customer focus icons
├── oncall/          # On-call management icons
├── contact/         # Contact method icons
└── tools/           # Development tool images (optional)
```

## Customizing Icons

All SVG files are simple and can be easily edited:

1. Open the SVG file in any text editor
2. Modify colors, shapes, or paths as needed
3. Or replace with your own SVG designs
4. Maintain the same filename to avoid updating HTML

## Icon Sizes

Icons are automatically scaled by CSS:
- Technology badges: 48x48px
- System cards: 48x48px
- Challenge cards: 56x56px
- Leadership cards: 48x48px
- Strength icons: 24x24px
- On-call highlights: 24x24px
- Contact icons: 32px32px
